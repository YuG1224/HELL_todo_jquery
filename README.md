# HELL_todo_jquery
地獄のようなjQuery

## 開発背景
何も考えずに開発すると辛くなることを実感してほしい

## 機能
* todoをサーバーから取得し、表示する機能
* todoをサーバーに投稿する機能
* todoの達成をチェックする機能し、サーバーを更新する機能
* 達成しているtodoの見栄えを変える機能
* 残っているtodoのみにフィルタリングをかける機能
* 残りtodo数をカウントする機能
